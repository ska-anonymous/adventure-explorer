Adventure Explorer App

The Travel Destinations app is a mobile application that allows users to explore and save travel destinations. It provides a user-friendly interface with three main screens: New Destination, Destinations List, and About Me. The app fetches data from the Zippopotam API to provide detailed information about places based on user input.

Features

•	New Destination Screen: Enter country, state, and city names to search for places.
•	Destinations List Screen: View and manage a list of saved travel destinations.
•	About Me Screen: Learn more about the developer and their background.

Getting Started

To get started with the Travel Destinations app, follow these steps:

1.	Download the app files in zip.
2.	Install dependencies: `npm install` or `yarn install`
3.	Run the app: `npm start` or `yarn start`
4.	Connect your device or emulator to view the app.

Technologies Used

•	React Native: A JavaScript framework for building mobile applications.
•	Expo: A platform for developing and deploying React Native apps.
•	SQLite: A lightweight database for storing and managing travel destination information.
•	Zippopotam API: An API service used to retrieve detailed place information.
